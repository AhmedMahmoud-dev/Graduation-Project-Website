# Emotra: AI-Powered Emotion Intelligence Platform

## Overview
**Emotra** is a state-of-the-art emotional intelligence platform designed to analyze and map human sentiments across multiple modalities. Built as a comprehensive graduation project, it leverages modern machine learning architectures to provide deep insights into how emotions evolve in different contexts.

## The Core Idea
The central idea behind Emotra is to bridge the gap between raw digital data (text, audio, image, video) and human emotional understanding. By applying "scientific precision" to sentiment analysis, the platform allows users to decode complex emotional journeys and patterns that are often invisible to the naked eye.

## Key Features & Components

### 1. Multimodal Emotion Analysis
The platform is designed to handle four primary input types:
*   **Text Analysis (Available):** Uses fine-tuned transformer models to detect emotional content and sentiment in written text.
*   **Audio Analysis (Available):** Tracks emotional intensity and sentiment through voice prosody, speech patterns, and acoustic cues.
*   **Image Analysis (Coming Soon):** Leverages computer vision to detect facial expressions and emotional cues in static images.
*   **Video Analysis (Coming Soon):** Provides real-time emotion tracking across video frames for dynamic sentiment mapping.

### 2. Specialized Pipelines: Ideology Classification
Beyond general emotion, the project includes specialized logic for **Political Ideology Classification**. This sub-feature analyzes text to categorize it into political leanings (e.g., Left, Center, Right), helping users understand the ideological bias in media and communication.

### 3. User Experience & Dashboard
*   **Personal Dashboard:** A central hub for users to view their analysis statistics and recent activities.
*   **History & Archive:** Users can revisit past analyses to track changes over time.
*   **Comparative Analysis:** Tools to compare emotional profiles across different inputs or time periods.
*   **Modern UI/UX:** A premium, dark-themed interface built with Angular and Tailwind CSS, prioritizing visual excellence and smooth interactions.

### 4. Admin & Oversight
*   **Admin Dashboard:** Provides system-wide oversight, user management, and performance monitoring.
*   **Model Management:** Infrastructure to handle various ML models and their configurations.

## How It Helps Users
Emotra serves a wide range of needs by providing actionable emotional data:
*   **Self-Awareness:** Individual users can analyze their own communications or recordings to understand their emotional patterns.
*   **Media Analysis:** Helps in identifying the emotional tone and ideological bias in news articles and social media posts.
*   **Customer Insights:** Businesses can use the platform to understand the emotional subtext in customer feedback or voice recordings.
*   **Research & Education:** Provides a scientific tool for studying human behavior and sentiment evolution.

## Technical Stack
*   **Frontend:** Angular 19+ with Tailwind CSS for a responsive, premium UI.
*   **Backend/ML Logic:** Python-based services utilizing Transformers, NLTK, and custom neural architectures.
*   **Persistence:** SQL-based database for tracking analysis history and user data.
*   **Deployment:** Optimized for modern cloud environments with Vercel support.
